## ⚠️ Known Issues & Required Fixes

The `new_home_page.dart` has 7 errors that need fixing to match your existing API:

### Issue 1: getMeditations API
**Error**: The `limit` parameter doesn't exist
**Location**: Line 96

**Current Code**:
```dart
final meditations = await _supabaseService.getMeditations(limit: 1);
```

**Fix Options**:
1. Remove limit and take first element:
```dart
final allMeditations = await _supabaseService.getMeditations();
final meditations = allMeditations.isNotEmpty ? [Meditation.fromMap(allMeditations.first)] : <Meditation>[];
```

2. Or query directly:
```dart
final response = await SupabaseService.instance.client
    .from('meditations')
    .select()
    .limit(1);
final meditations = response.map((m) => Meditation.fromMap(m)).toList();
```

### Issue 2: getMeditations Return Type
**Error**: Returns `List<Map<String, dynamic>>` not `List<Meditation>`
**Location**: Line 101

**Fix**:
```dart
// Change from:
final meditations = await _supabaseService.getMeditations();

// To:
final meditationsData = await _supabaseService.getMeditations();
final meditations = meditationsData.map((m) => Meditation.fromMap(m)).toList();
```

### Issue 3: createMoodEntry API
**Error**: Wrong method signature - expects `MoodEntry` object
**Location**: Lines 143-147

**Current (Wrong)**:
```dart
await _supabaseService.createMoodEntry(
  userId: user.id,
  moodLevel: mood,
  note: '',
  activities: [],
);
```

**Fix**:
```dart
await _supabaseService.createMoodEntry(
  MoodEntry(
    entryId: '', // Auto-generated by DB
    userId: user.id,
    moodLevel: mood,
    note: '',
    timestamp: DateTime.now(),
    emotionFactors: [],
    tags: [],
  ),
);
```

### Issue 4: MoodProvider.refreshMoods()
**Error**: Method doesn't exist on MoodProvider
**Location**: Line 165

**Fix Options**:
1. Remove this line (mood will refresh on next page visit)
2. Or add a refresh method to MoodProvider
3. Or directly call: `context.read<MoodProvider>().loadMoodEntries(user.id);`

### Issue 5: Deprecated withOpacity
**Info**: 5 occurrences of deprecated `.withOpacity()` in new_home_page.dart

**Fix**: Replace all with `.withValues(alpha: x)`:
```dart
// Find and replace:
```

## 📦 Components Overview

### 1. NeumorphicCard
**Purpose**: Base component for all cards

**Key Features**:
- Soft box shadows (dual-layer)
- Pressed state animation (150ms)
- Customizable padding & color
- Tap callback support

**Usage**:
```dart
NeumorphicCard(
  onTap: () => print('Tapped'),
  child: Text('Content'),
)
```

### 2. MoodQuickCheck
**Purpose**: One-tap mood logging

**Key Features**:
- 5-point emoji scale
- Hover & selection states
- Smooth animations (200ms, ease-out-back)
- Color-coded by mood
- Accessible (60px+ height)

**Usage**:
```dart
MoodQuickCheck(
  onMoodSelected: (mood) => _handleMoodLog(mood),
  currentMood: 4, // Optional pre-selection
)
```

### 3. WellnessStatsCard
**Purpose**: Display streak statistics

**Key Features**:
- 3 stats: Current / Longest / Total
- Color-coded values
- Tap to navigate
- Visual dividers

**Usage**:
```dart
WellnessStatsCard(
  streak: _streakData,
  onTap: () => Navigator.push(...),
)
```

### 4. FeaturedMeditationCard
**Purpose**: Showcase meditation

**Key Features**:
- Gradient header
- Duration badge
- Category tag
- 2-line text truncation
- Play icon affordance

**Usage**:
```dart
FeaturedMeditationCard(
  meditation: _meditationObj,
  onTap: () => Navigator.push(...),
)
```

### 5. QuickActionGrid
**Purpose**: Fast navigation

**Key Features**:
- 2-column responsive grid
- Press state feedback
- Icon + label
- Color-coded actions

**Usage**:
```dart
QuickActionGrid(
  actions: [
    QuickActionItem(
      title: 'Experts',
      icon: Icons.psychology,
      color: AppColors.primaryLight,
      onTap: () => Navigator.push(...),
    ),
    // ... more actions
  ],
)
```
